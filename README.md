# Girls Higher Secondary School (GHSS) Portal

A modern, responsive education portal for Girls Higher Secondary School, built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Branded UI**: Custom school logo, tagline, and theme.
- **Hero Section**: High-impact hero with admissions CTA.
- **School Gallery**: Categorized gallery (Campus, Events, Sports, Labs).
- **Testimonials**: Interactive slider with feedback from students, parents, and alumni.
- **Contact System**: Comprehensive contact form and Google Maps integration.
- **SEO Ready**: Metadata, OpenGraph, Twitter Cards, Sitemap, and Robots.txt included.
- **Performance**: Optimized images using `next/image` and lazy-loaded sections.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Page routes and layouts.
- `src/components/layout`: Global components like Navbar and Footer.
- `src/components/sections`: Home page sections.
- `src/components/ui`: Reusable UI components.
- `public`: Static assets.
